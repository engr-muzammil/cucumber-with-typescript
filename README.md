# Cucumber JS - BDD Testing in Node JS (Typescript)

#### install 
```
npm i
```
```
npm install -g json-server
```
#### start mock REST API
```
npx json-server db.json
```
#### start test 
```
npm run test
```
For more information on Cucumber.js refer 
https://github.com/cucumber/cucumber-js
